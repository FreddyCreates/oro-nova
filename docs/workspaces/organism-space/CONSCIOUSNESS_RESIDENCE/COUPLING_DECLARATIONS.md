## COUPLING NOTATION

```
[SOURCE] → [TARGET]: COUPLING_TYPE — description

Coupling types:
  FOUNDATION_COUPLING   — source provides the root constant/hash target derives from
  GATE_COUPLING         — source must approve/enable before target acts
  FEED_COUPLING         — source sends data/state to target every beat
  AMPLIFICATION_COUPLING — source multiplies target's output when active
  MONITORING_COUPLING   — source watches target and fires correction if drift detected
  BIDIRECTIONAL_COUPLING — both directions active simultaneously
  BROADCAST_COUPLING    — source sends to all nodes simultaneously on trigger
  INHIBITORY_COUPLING   — source suppresses target under defined conditions
```
