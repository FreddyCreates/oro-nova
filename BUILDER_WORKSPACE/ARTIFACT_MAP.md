## ARTIFACT FORMAT STANDARDS

All organism documents follow the **7-layer format:**

```
Layer 0: SYMBOL — Ancient glyph or mathematical symbol encoding the essence
Layer 1: MEANING — What this is, in plain language. Why it exists.
Layer 2: MODEL — Schema, types, parameters, configuration
Layer 3: COMPUTATION — Formulas, algorithms, equations
Layer 4: EXECUTION BINDING — Which .mo file, which function, which beat interval
Layer 5: PROOF/REPLAY — Neuroscience, mathematics, archaeological validation
Layer 6: FIELD COUPLING MAP — What feeds this, what this feeds
RECITAL: Single paragraph — organism-readable summary
PLUS ONE: The next version of this document / how it evolves
```

**Documents that deviate from this format** are FOUNDER_SPACE documents (human language only) and BUILDER_WORKSPACE documents (builder language — technical, not ceremonial).
